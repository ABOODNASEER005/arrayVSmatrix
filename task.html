ARRAY VS MATRICES

NumPy Arrays

Definition: A NumPy array (numpy.ndarray) is a versatile, n-dimensional array object that can represent vectors, matrices, and higher-dimensional tensors.

Creation: Created using np.array().

Flexibility: Can have any number of dimensions (1D, 2D, 3D, etc.).

Operations: Element-wise operations are the default. Matrix multiplication requires np.dot() or the @ operator.

Broadcasting: Supports broadcasting for operations on arrays of different shapes.

Usage: Preferred for most numerical computations due to its flexibility and broader functionality.

NumPy Matrices

Definition: A NumPy matrix (numpy.matrix) is a specialized 2D array specifically designed for linear algebra operations.

Creation: Created using np.matrix().

Flexibility: Restricted to 2 dimensions only.

Operations: The * operator performs matrix multiplication. Element-wise operations require np.multiply().

Broadcasting: Does not support broadcasting as comprehensively as arrays.

Usage: Less commonly used today, as arrays are more general and flexible. The numpy.matrix class is largely deprecated.
